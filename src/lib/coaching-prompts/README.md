# Coaching Prompts Directory

This directory contains custom coaching prompts and expertise modules.

## File Structure

- `chatgpt-coach-prompt.md` - Your original ChatGPT coach prompt
- `olympic-expertise/` - Specialized Olympic-level coaching modules
- `custom-personas/` - Different coaching personalities
- `sport-specific/` - Sport-specific adaptations

## How to Add Your Prompt

Save your ChatGPT coach prompt as:
```
/Users/jasontraum/stride-os/src/lib/coaching-prompts/chatgpt-coach-prompt.md
```

Or any of these formats:
- `.md` - Markdown (recommended)
- `.txt` - Plain text
- `.json` - Structured format

## Integration

Once saved, we can:
1. Parse and integrate your coaching philosophy
2. Extract key methodologies
3. Build custom expertise modules
4. Create hybrid prompts combining best practices
